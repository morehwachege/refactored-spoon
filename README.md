# Weather Client Application

A weather application client to search for weather information about major cities.


## API Information

#### Get city information

```http
  GET /${city}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `none` | **None Required** |
